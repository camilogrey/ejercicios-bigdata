# Entregas: Streaming y Cloud (Modulos 08-09)

Esta carpeta contiene las entregas de los modulos opcionales avanzados.

## Estructura

```
streaming_cloud/
├── README.md                    ← Este archivo
│
├── apellido_nombre_1/           ← Entrega del alumno 1
│   ├── PROMPTS.md               ← Documentacion de prompts (OBLIGATORIO)
│   ├── kafka/                   ← Modulo 08 (si lo completo)
│   └── localstack/              ← Modulo 09 (si lo completo)
│
├── apellido_nombre_2/           ← Entrega del alumno 2
│   └── ...
│
└── ...
```

## Como Entregar

1. Crea tu carpeta: `mkdir apellido_nombre`
2. Copia la plantilla de PROMPTS.md (ver guia en docs/)
3. Documenta tus prompts reales
4. Agrega capturas de pantalla
5. Sube con git push

## Guia Completa

Ver documentacion: [Entregas Streaming/Cloud](https://todoeconometria.github.io/ejercicios-bigdata/entregas/streaming-cloud/)

## Que se Evalua

- **PROMPTS.md** - Tus prompts reales (40%)
- **Retos completados** - Codigo funcional (30%)
- **Capturas** - Evidencia visual (15%)
- **Reflexion** - Comprension (15%)

**Nota:** Estos modulos son OPCIONALES y dan bonus sobre el Trabajo Final.
